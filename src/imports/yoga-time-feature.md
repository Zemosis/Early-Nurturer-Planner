Add Yoga Time to Daily Schedule

Module: Daily Schedule

Add a dedicated Yoga Time block to the daily schedule so teachers/caretakers can clearly see and manage this activity as part of the structured day.

🧘 YOGA TIME – DAILY SCHEDULE INTEGRATION
1️⃣ Add New Schedule Block

Create a new activity block labeled:

🧘 Yoga Time

Include:

Time stamp (editable)
Example: 10:30 AM – 10:45 AM

Short description (optional)

Linked to Circle Time Yoga module

2️⃣ Editable Time Slot

Teachers can adjust start/end time

Follows the same timestamp format as other activities

Auto-adjusts schedule spacing dynamically

3️⃣ Visual Differentiation

Give Yoga Time its own distinct color (calm but noticeable):

Suggested options:

Soft lavender

Light teal

Calm blue

Must be clearly different from:

Green (Outdoor)

Yellow (Art)

Other existing schedule colors

4️⃣ Interaction Behavior

When clicking Yoga Time:

Options:

Open Yoga Section directly

Edit pose sequence

Adjust timer length

Switch Auto / Manual mode

5️⃣ Mobile Layout

Vertical schedule layout

Yoga appears as its own rounded card

Clear timestamp at top

Icon displayed next to label

Minimum tap height: 44px

6️⃣ Desktop / iPad Layout

Appears within daily schedule grid

Consistent sizing with other schedule blocks

Maintains color separation

Clean alignment

🔄 Optional Enhancement

Allow Yoga Time to be:

Toggled on/off

Automatically added when “Generate Week” is clicked

Draggable to reorder within the schedule

🎯 FINAL RESULT

Yoga Time should feel:

✔ Intentionally placed in the day
✔ Clearly visible
✔ Easy to edit
✔ Connected to Circle Time module
✔ Structured but flexible
