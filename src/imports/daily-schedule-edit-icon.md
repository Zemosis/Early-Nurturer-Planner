🎨 FEATURE UPDATE REQUEST
Make Edit (Pen) Icon Always Visible – Mobile

Module: Daily Schedule Tab

🎯 GOAL

Ensure the Edit (Pen) icon is always visible on the mobile version of the Daily Schedule.

Currently, the icon may:

Only appear on hover (not ideal for mobile)

Be hidden inside a menu

Be partially clipped

Disappear when scrolling

This creates friction for teachers who need quick editing access.

✅ REQUIRED CHANGES
1️⃣ Always Visible on Mobile

Remove hover-only behavior (mobile has no hover)

Display the pen icon persistently on each schedule card

Position it clearly in the top-right corner of each activity block

2️⃣ Proper Placement

Inside each schedule card:

Title (left)

Time stamp (below title)

Pen icon (top-right)

Icon should:

Not overlap text

Not be clipped

Have at least 12–16px padding from edges

3️⃣ Touch-Friendly Design

Minimum tap target: 44px × 44px

Add subtle background circle behind icon for clarity

Use soft contrast color

Add light tap feedback (scale or color change)

4️⃣ Scrolling Behavior

Icon must stay attached to its card

Should not float incorrectly

Should not disappear during scroll

Should not overlap other UI elements

5️⃣ Visual Hierarchy

Edit icon should:

✔ Be visible
✔ Be easy to tap
✔ Not overpower the card
✔ Feel integrated into design

Suggested style:

24px icon

32–36px circular touch area

Light background fill (10–15% opacity of theme color)

🎯 EXPECTED RESULT

Teachers should be able to:

✔ Immediately see how to edit
✔ Tap quickly without hunting for controls
✔ Modify schedule smoothly

No hidden actions.
No hover dependency.
No clipped icons.
